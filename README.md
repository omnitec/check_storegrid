check_storegrid
===============

Nagios plugin for checking the status of StoreGrid backups


License:		GPLv2

Created by:

HELP WANTED!

The purpose of this plugin is to allow Nagios to monitor the backup status 
of each backup job running on a StoreGrid Backup Controller.
